PIZZA CAT BLACKJACK

This app is a fun take on the classic casino card game, blackjack. Its theme is based on the popular internet meme, pizza cat. Try your luck playing against the dealer, and try to win as many pizzas as you can before running out. Turn your sound on, as pizza cat makes noises when you click on the game's buttons.

STACK:   HTML  |   CSS  |   JAVASCRIPT  |   JQUERY



WANDERSHARE

Wandershare is an app that allows users to post information and images about their travels in a way that’s instantly sharable and digestible. Users can log in, and see what trips friends have taken and click to learn more about each trip, including: dates traveled, trip itinerary, recommended packing list, and more.

STACK:   HTML   |   CSS   |   JAVASCRIPT   |   JQUERY   |   NODEJS   |   MONGODB



GIF.SY

GIF.SY is a full CRUD application that allows users to search and select their favorite GIFs, save them to a collection, and share them via Facebook

STACK:   HTML   |   CSS   |   JAVASCRIPT  | JQUERY   |   NODEJS   |   MONGODB   |   GIPHY API   |   ANGULAR



VOLUNTEER MANAGEMENT APP

This Volunteer Management App allows any organization (New York Cares, for example) to manage both their volunteer projects and volunteers in one place. In just a couple of clicks, an administrator can add, edit, and remove both projects and volunteers, add projects to volunteers and/or volunteers to projects, as well as add notes about each project as they come up.

STACK:   HTML   |   CSS   |   RUBY ON RAILS   |   POSTGRESQL



WEEKENDER

Weekender is a single-page app that allows users to explore NYC at the push of a button. Once users decide to choose a neighborhood or have the app randomly select a neighborhood, Weekender then generates a full weekend day including: a place for brunch, a nearby activity, a cool spot for drinks, and dinner, and instant directions to each stop from his or her current location. Finally, the user can opt to save the day in his or her “days”, delete a day, or share to Facebook.

STACK:   HTML   |   CSS   |   JAVASCRIPT   |   AJAX   |   GOOGLE MAPS API   |   YELP API   |   ANGULAR   |   NODEJS   |   MONGODB
